# @xtaskjs/common

Common utilities for xtaskjs, a modern, fast Node.js web framework.

## Installation
```bash
npm install @xtaskjs/common
```

## Usage
```typescript
import { ... } from '@xtaskjs/common';
```

## Features
- Logger
- Decorators
- Shared types and constants

## Documentation
See [xtaskjs.com](https://xtaskjs.com) for full documentation.
